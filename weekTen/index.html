Client-Side form Validation

Client-side validation is essentially a pre check before a form is submitted. This allows for the the data input to be checked and changed without having to be submitted. This creates a better user experience.

Obviously, when it comes to convenience, using built-in form validation is preferred. It seems to be a little simpler and easier to write.

I really like the ability to style elements accourding to their validity. I like the user experience that this provides.
